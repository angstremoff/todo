## Краткая память по проекту

- Стек: React Native 0.73.9, TypeScript. Локальная БД SQLite (`react-native-sqlite-storage`).
- Задачи: текст + статус (`active`/`done`), привязаны к рабочему пространству (workspace).
- Рабочие пространства: создаются/переименовываются/удаляются; меню действий через кнопку «⋯» на таблетке.
- Вкладки задач: «Все» (активные + архив), «Текущие», «Архив».
- Задачи: меню «⋯» на карточке (редактирование, удаление), кнопка статуса «Готово/Вернуть».
- Темы: светлая/тёмная, переключатель в правом верхнем углу.
- Импорт/экспорт: кнопки «Экспорт»/«Импорт» рядом с заголовком рабочих пространств. Экспорт выдаёт JSON (копирование/поделиться), импорт принимает JSON; режимы: «Добавить» (merge) или «Заменить» (replace).
- Иконка: адаптивные ресурсы `ic_launcher_foreground`/background.
- Тесты: `npm test -- --runInBand`.
- Сборка релиза: `cd android && ./gradlew assembleRelease -x lint --no-daemon`; APK копируется в `~/Desktop/todo-<version>-release.apk`.
